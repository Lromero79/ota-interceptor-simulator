**Educational simulation tool for understanding OTA (Over-The-Air) call, text, and media interception concepts.**

> **Important**: This is a **simulation-only** app for learning and research in controlled environments. Real interception tools may violate laws.

## Features
- Realistic simulation of incoming calls, SMS, and media streams
- Cross-platform support (Android, iOS, Linux, macOS)
- Background service for monitoring
- Interactive map for location visualization
- Log export and troubleshooting guide
